# datasheet.md# Datasheet for Dataset, Simulated Mango Recalcitrant Seed Drying Physiology## 1. Dataset summary**Dataset name:** Simulated Mango Recalcitrant Seed Drying Physiology Dataset  **Dataset type:** Synthetic, generated from parametric simulation + stochastic noise  **Data unit:** One row per replicate per treatment per time point  **Purpose:** Provide realistic-looking time-course data for analysis of recalcitrant seed dehydration injury.## 2. MotivationRecalcitrant seeds cannot be stored like orthodox seeds, and experiments often quantify:- moisture loss kinetics,- germination decline near a critical moisture content,- electrolyte leakage (conductivity),- respiration changes.This dataset is designed to support:- method development,- training,- and experimental planning.## 3. Data collection processNo biological specimens were used. Data were generated by a deterministic simulator with random noise.### Experimental factors simulated- Drying modality: air drying vs sun drying- Endocarp status: with vs without endocarp- Time points: 0ﾐ104 hours- Replicates: 3 per condition (configurable)## 4. Contents### Files- `sim_outputs/mango_recalcitrant_drying_simulated.csv`- `sim_outputs/cmc_logistic_fits.csv`- `sim_outputs/time_to_threshold_metrics.csv`### Main dataset columns- `treatment`: ADWE, ADWO, SDWE, SDWO- `drying`: AD or SD- `endocarp`: WE or WO- `time_h`: time in hours- `rep`: replicate number- `moisture_pct`: moisture content (% FW basis), simulated- `germination_pct`: germination percent (0ﾐ100), simulated- `conductivity_uS`: normalized conductivity proxy, simulated- `respiration_mgCO2_g_h`: respiration proxy scale, simulated- `viable`: derived label (1 if germination ｳ 50%, else 0)- `PC1`, `PC2`: PCA scores computed during analysis- `cluster`: injury-state cluster id from KMeans (analysis-derived)## 5. Preprocessing and cleaningNo cleaning is required beyond standard handling of numeric columns.- Ensure correct dtype for categorical variables if needed.- Some values are clipped to physiological bounds (e.g., germination 0ﾐ100).## 6. Recommended uses- Demonstrate curve fitting (CMC estimation)- Practice regression modeling of factorial designs- Develop visualization panels for manuscript-style reporting- Teach multivariate embedding (PCA) and clustering## 7. Uses to avoid- Reporting simulated values as experimental results- Using outputs to set real storage thresholds without calibration- Using the dataset as a substitute for cultivar-specific validation## 8. DistributionThis dataset can be shared openly if labeled as synthetic.Suggested license: CC BY 4.0 for data, MIT for code.## 9. Maintenance- Document any parameter changes (CMC, drying rates, noise) and bump version.- Keep a changelog of simulator updates.## 10. Known limitations and biases- Parameters reflect literature-informed behavior but are not cultivar-validated.- Moisture is treated as a bulk variable, not a spatial gradient within seed tissues.- Conductivity and respiration are proxies unless matched to a specific lab assay.
